	\begin{flushleft}
		\textbf{RESUMEN:} \\
		\justifying{RS Chat es una aplicación web para la interacción entre estudiantes y docentes en tiempo real.
		Su objetivo es facilitar la comunicación y colaboración entre los usuarios. Ofrece un sistema de almacenamiento
		del historial de mensajes, notificaciones y compartición de contenido multimedia. Para este se realiza un
		filtrado de imágenes, mediante el uso de inteligencia artificial, para evitar que en el chat se envíe contenido
		inapropiado. El sistema de insignias permite a los usuarios obtener recompensas por su participación en la
		aplicación, lo que les motiva a utilizarla. La aplicación se ha desarrollado utilizando buenas prácticas de
		programación, con diversos patrones de diseño y frameworks de desarrollo. Esto permite una fácil escalabilidad
		y mantenimiento de la misma, así como la posibilidad de añadir nuevas funcionalidades de manera sencilla y
		eficiente. Utilizando el despliegue mediante Docker, se ha conseguido una fácil instalación y configuración de
		la aplicación, así como una mayor portabilidad. La integración continua y tests de código empleados han
		permitido una mayor eficiencia y calidad del código, lo que se traduce en un menor tiempo de desarrollo y una
		mayor robustez del producto final.}
	\end{flushleft}

	\begin{flushleft}
		\textbf{ABSTRACT:} \\
		\justifying{RS Chat is a web application for real time interaction between students and teachers.
		Its objective is to facilitate communication and collaboration between users. It offers a message history
		storage system, notifications and multimedia content sharing. For this, an image filtering is performed, using
		artificial intelligence, to prevent inappropriate content from being sent in the chat. The badge system allows
		users to obtain rewards for their participation in the application, which motivates them to use it. The
		application has been developed using good programming practices, with various design patterns and development
		frameworks. This allows easy scalability and maintenance of the application, as well as the possibility of
		adding new functionalities in a simple and efficient way. Using the deployment through Docker, an easy
		installation and configuration of the application has been achieved, as well as greater portability. The
		continuous integration and code tests used have allowed greater efficiency and quality of the code, which
		translates into a shorter development time and a higher robustness of the final product.}
	\end{flushleft}
